# apps

This is a GitHub Pages repository hosting informational/landing pages for two apps: **preak** and **soundora**.

## Structure

```
preak/       - GitHub Pages site for the preak app
soundora/    - GitHub Pages site for the soundora app
```

Each app directory contains:
- `index.html` - main landing page
- `contact.html` - contact page
- `privacy-policy.html` - privacy policy page

## Tech

- Plain HTML/CSS only — no frameworks, no build tools, no JavaScript dependencies
- Static files served directly via GitHub Pages

## Conventions

- Commit messages use conventional commits: `feat:`, `chore:`, `fix:`, etc.
- Each app is fully independent — do not share code or assets between them
- Do not add new files unless explicitly requested
